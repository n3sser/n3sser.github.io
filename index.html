This is Guide.

The application is a personal electronic audio guide for the time being which is used for tourist places all around Oman. The user can use the application individually on his/ her device. The application allows the user to follow up the visited place easily by scanning Qr Codes which are distributed around the place.

The application allows you to enjoy your tour with a detailed audio description with sound effects and 3D sound system. The application is available in four languages: Arabic, English, German and French.
